Maéir Trinkets
Premium E-Commerce Storefront & Headless CMS

Lead Full-Stack Developer

Designed and developed a fully custom, high-end e-commerce platform for a premium jewelry brand. The architecture bridges a luxury customer-facing storefront with a secure, custom-built Admin Dashboard ("Maéir Vault"), allowing real-time inventory management without code.

Core Tech Stack

Next.js & React TypeScript Tailwind CSS Airtable API (CMS) Cloudinary API Brevo API Vercel Edge Deploy

Customer Storefront Highlights

  • Premium UI/UX: Luxury aesthetic leveraging glass-morphism, custom 3D perspective tilts, staggered floating polaroids, and dynamic light glares.
  • Automated WhatsApp Checkout: Engineered a custom shopping cart that parses item data, calculates district-based delivery logic, and formats the order into a perfectly structured WhatsApp transmission for instant closing.
  • Dynamic Inventory: Live API pulling of product statuses, auto-flagging sold-out sizes, and updating categories dynamically.
  • Interactive Social Proof: Integrated floating visual reviews that expand into full-screen lightboxes.

The "Maéir Vault" Admin Dashboard

  • Secure Operations: Built a PIN-locked administrative entryway for team members.
  • Real-Time Data Control: Total CRUD capability. Admins can update pricing, tag bestsellers, and manage stock sizes live.
  • Media & Cache Management: Direct Cloudinary image uploads linked to the Airtable DB. Built-in Vercel revalidatePath hooks to instantly wipe the site cache when the dashboard is updated.

The Engineering Solution

The client required a bespoke luxury feel but refused the rigid structure and high monthly fees of Shopify. By decoupling the frontend (Next.js) from a Headless CMS (Airtable), I delivered a lightning-fast, zero-platform-fee storefront paired with a dashboard that makes managing hundreds of SKUs as simple as editing a spreadsheet.
